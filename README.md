### вҡҷпёҸ Optimization Techniques
- **Inline Critical CSS** вҖ” above-the-fold styles load instantly with zero render-blocking
- **Async Font Loading** вҖ” Google Fonts (`Orbitron`, `Inter`) load asynchronously via `media="print"` swap
- **Deferred Stylesheets** вҖ” non-critical CSS is deferred until after first paint
- **GPU-Accelerated Animations** вҖ” `will-change` and `transform` ensure smooth 60fps effects
- **Lazy Rendering** вҖ” game hub tiles load progressively using IntersectionObserver for near-instant page load
